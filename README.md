## Run Locally

Clone the project

```bash
  git clone https://link-to-project](https://github.com/SantiArandaRod/dev_CreacionesMechasStore)
```

Go to the project directory

```bash
  cd dev_CreacionesMechasStore
```

Install requirements

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  uvicorn main:app
```
